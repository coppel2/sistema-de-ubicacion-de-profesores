# sistema-de-ubicacion-de-profesores
Los estudiantes y administrativos frecuentemente desconocen en qué aula o edificio se encuentra un profesor.  -El horario institucional existe, pero no está centralizado ni actualizado en tiempo real.  -No se registran cambios como juntas, ausencias o cambios de aula y esto provoca confusión, pérdida de tiempo y mala comunicación.
